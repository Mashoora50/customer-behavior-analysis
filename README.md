# customer-behavior-analysis
Data analytics project showing customer behavior analysis  using Python,SQL, power BI.

Project Overview
This project analyzes customer shopping behavior using Python, SQL, MySQL, and Power BI to uncover valuable business insights related to customer spending patterns, product performance, customer segmentation, subscriptions, discounts, and revenue trends.

The workflow combines:

Data Cleaning & Transformation using Python (Pandas)
Database Storage & Querying using MySQL
Exploratory Data Analysis (EDA) using SQL
Interactive Dashboard Development using Power BI

Business Questions:

This project answers several key business questions:

What is the total revenue generated by male and female customers?
Which customers used discounts but still spent above the average purchase amount?
What are the top 5 highest-rated products?
What are the lowest-rated products?
How does average spending differ between Standard and Express shipping?
Do subscribed customers spend more than non-subscribed customers?
Which products have the highest percentage of discounted purchases?
How can customers be segmented into New, Returning, and Loyal customers?
What are the top 3 most purchased products within each category?
Are repeat buyers more likely to subscribe?
Which age groups generate the highest revenue?
